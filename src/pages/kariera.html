W związku z dynamicznym rozwojem firmy poszukujemy specjalistów, którzy dołączą do naszego zespołu. Możesz aplikować na jedno z dostępnych stanowisk:

Senior Frontend Developer

Wymagania:

  • bardzo dobra znajomość HTML5, CSS/SCSS i JS oraz standardów W3C,
  • doświadczenie z RWD na różnych przeglądarkach / urządzeniach,
  • bardzo dobra znajomość Vue.js / React,
  • znajomość narzędzi takich jak webpack, npm, yarn, gulp,
  • doświadczenie w pracy z PHP, wykorzystywanie Gita oraz konsoli w codziennej pracy,
  • dodatkowym atutem będzie doświadczenie z rozwiązaniami headless, PWA, platformami TYPO3

Symfony PHP Developer

Wymagania:

  • znajomości PHP 7 (OOP, MVC, dependency injection)
  • minimum 2 lata zawodowego doświadczenia z PHP
  • bardzo dobrej znajomości Symfony
  • znajomości Gita
  • znajomości baz danych SQL (MySQL i PostgreSQL)
  • znajomości języka angielskiego w stopniu komunikatywnym

Node.js Developer

Wymagania:

  • znajomość JavaScript i/lub TypeScript (MVC, dependency injection)
  • minimum 3 lata zawodowego doświadczenia w programowaniu
  • znajomość Express.js, Nest.js lub Serverless Framework i doświadczenie w jednym z tych frameworków
  • znajomość Gita
  • znajomość baz danych SQL (MySQL i PostgreSQL) oraz MongoDB
  • dodatkowe atuty: znajomość AWS i chmury Google

React Native Developer

Wymagania:

  • znajomość JavaScript na potrzeby tworzenia zaawansowanych aplikacji cross-platformowych
  • minimum 2 lata zawodowego doświadczenia w tworzeniu aplikacji mobilnych z wykorzystaniem React Native
  • praktyczna znajomość React.js
  • doświadczenie w integracji bibliotek Swift/Objective-C/Java z React Native
  • doświadczenie we wdrażaniu aplikacji mobilnych do App Store/Google Play
  • znajomość narzędzi automatyzujących CI/CD (Fastlane, Bitrise, CircleCI)
  • znajomości języka angielskiego w stopniu komunikatywnym
